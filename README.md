# Mongolia

A Magnolia CMS utilities library

### Generating JavaDoc

To generate the JavaDoc you need to use the following command.

```
mvn javadoc:javadoc
```

It will generate `target/site/apidocs/`. To navigate the docs, just open `index.html`.
